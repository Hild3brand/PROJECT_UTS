PHP SEDERHANA

Input

Proses

Hasil

Perbedaan post ($_POST) dan get ($_GET)

Method POST akan mengirimkan data atau nilai langsung ke action untuk ditampung, tanpa menampilkan pada URL.
FORM → ACTION

Sedangkan method GET akan menampilkan data/nilai pada URL, kemudian akan ditampung oleh action.
FORM → URL → ACTION

FORM

tag : <form>
Untuk mengambil informasi.

<form action="prosess.php" method="GET">
<!-- form field di sini -->
</form>

Action : dapat diisi dengan halaman URL dari endpoint yang akan memproses form.


CONTOH :

<!DOCTYPE html>
<html>
<head>
   <title>Form Login</title>
</head>

<body>
   <form action="login.php" method="POST">
     <fieldset>
     <legend>Login</legend>
     <p>
       <label>Username:</label>
       <input type="text" name="username" placeholder="username..." />
     </p>
     <p>
       <label>Password:</label>
       <input type="password" name="password" placeholder="password..." />
     </p>
     <p>
       <label><input type="checkbox" name="remember" value="remember" /> Remember me</label>
     </p>
     <p>
       <input type="submit" name="submit" value="Login" />
     </p>
     </fieldset>
   </form>

</body>
</html>

HASIL :



Login

Fieldset = membuat garis luar
Legend = untuk memberi teks pada fieldset

Type
Text = input tipenya text
Password = inputnya menjadi bulat-bulat
Checkbox = untuk membuat checkbox
Submit = untuk meng-submit

Berbagai value input type :

  • Number
  • Range
  • File
  • Image
  • Color
  • Data
  • Datetime-local
  • Time
  • Month
  • Week
